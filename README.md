# Customer Management REST API

A Spring Boot application that provides RESTful APIs to perform CRUD operations on Customer data.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL

## Features
- Create Customer
- Get Customer by ID
- Get All Customers
- Update Customer
- Delete Customer
- Layered architecture (Controller, Service, Repository)

## API Endpoints
- GET /customer
- GET /customer/{id}
- POST /customer
- PUT /customer/{id}
- DELETE /customer/{id}

## Tools Used
- Eclipse IDE
- Git & GitHub
- Postman
